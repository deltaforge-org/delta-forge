# Delta Forge

Binary releases and install instructions for the Delta Forge community edition.

- Website: [deltaforge.org](https://deltaforge.org)
- Install guide: [deltaforge.org/install](https://deltaforge.org/install)
- Documentation: [deltaforge.org/docs](https://deltaforge.org/docs)
- Issues: report bugs and feature requests in the [Issues tab](https://github.com/deltaforge-org/deltaforge/issues) of this repository

## Packages

The community edition is distributed as four packages:

| Package              | Description                       |
| -------------------- | --------------------------------- |
| `deltaforge`         | Desktop application               |
| `deltaforge-cli`     | Command-line interface            |
| `deltaforge-mcp`     | Model Context Protocol server     |
| `deltaforge-compute` | Compute node                      |

## Install

See [deltaforge.org/install](https://deltaforge.org/install) for the full, up-to-date list of supported package managers and install commands for Windows, macOS, and Linux.

## Releases

Release binaries are attached to this repository's [Releases](https://github.com/deltaforge-org/deltaforge/releases), with SHA256 checksums and, where applicable, GPG signatures.
